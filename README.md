# Card War Game

トランプを使った「戦争」ゲームを Ruby で実装しました。  
Quest の Step1~Step4 すべての機能を実装しました。

## セットアップ

bundle install

## ディレクトリ構成

・実行ファイル：bin/main.rb  
・各種クラスファイル：lib/  
・テストファイル：test/

## 実行コマンド

`ruby bin/main.rb`

## 使用技術

・Ruby 3.3.6  
・Minitest（テスト用）  
・Rubocop（スタイルチェック）  
・active_support/core_ext/array(メゾットのみ使用)
